# Background Remover with rembg

This project provides a simple script to remove the background of images using the **rembg** library, which is a deep learning-based background removal tool.

## Features
- **Free to use** on Google Colab
- No API keys required
- Removes image backgrounds efficiently
- Supports images from Google Drive
